<h3 id="installation">2. Installation and Setup</h3>

* If you don't already have installed **python 3.10.x** or higher get it from [![](https://img.shields.io/badge/Python-3776AB?logo=python&logoColor=white)](https://www.python.org/downloads)


* **Clone** this **repository** using:
  ```console
  git clone https://github.com/LukasKrah/WoW-Dashboard-Client
  ```
  or get the **latest release** from [![](https://img.shields.io/github/release/Nilusink/2dGravitySim.svg)](https://github.com/LukasKrah/WoW-Dashboard-Client/releases)
  

* **Install** required **packages** using:
  ```console
  pip install -r requirements.txt
  ```
  

* **Run** the program by using:
  ```console
  python main.py
  ```
  or just run main.py with your favourite IDE


* If something doesn't work out feel free to contact me: [![](https://img.shields.io/badge/ProtonMail-8B89CC?logo=protonmail&logoColor=white)](mailto:https://mail.proton.me/?subject=WoW-Dashboard)
